# MOVI5 - Turismo en el Pacífico Colombiano

Plataforma web dedicada a la promoción y reserva de experiencias turísticas en el Pacífico Colombiano. MOVI5 conecta a los viajeros con vuelos regionales, transporte marítimo y hospedajes de lujo, con un fuerte enfoque en la sostenibilidad y el desarrollo comunitario.

## Características Principales

- **Diseño Responsivo Moderno**: Interfaz adaptada a móviles y escritorio, con menú lateral tipo "drawer" y efectos de vidrio (glassmorphism).
- **Catálogo de Destinos**: Información detallada sobre destinos clave como Juanchaco, Ladrilleros, La Barra, Piangüita y San Cipriano.
- **Sistema de Reservas**: Integración para reservar vuelos, lanchas y hospedajes.
- **Secciones Informativas**:
  - `principal.html`: Página de inicio con "hero section" inmersiva y destacados.
  - `nosotros.html`: Información corporativa y compromiso social.
  - `destinos.html`: Galería de destinos turísticos.
  - `hospedaje.html`: Selección de hoteles y hostales aliados.
  - `experiencias.html`: Planes y actividades turísticas.
  - `servicios.html`: Detalle de rutas aéreas, marítimas y terrestres.
  - `contacto.html`: Formularios, FAQ y contacto directo.

## Tecnologías Utilizadas

- **HTML5 Semántico**: Estructura sólida y accesible.
- **Tailwind CSS**: Utilizado vía CDN para estilizado rápido y consistente, con configuración personalizada para colores (Emerald, Forest, Earth) y fuentes.
- **JavaScript (Vanilla)**: Lógica para interacciones de UI, menú móvil y modales.
- **Google Fonts**: Tipografías 'Epilogue' y 'Playfair Display'.
- **Material Symbols**: Iconografía moderna de Google.

## Instrucciones de Uso

1. **Clonar el repositorio**:
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```

2. **Instalar dependencias** (si aplica para herramientas de desarrollo locales):
   ```bash
   npm install
   ```

3. **Ejecutar en desarrollo**:
   ```bash
   npm run dev
   ```
   O simplemente abrir el archivo `principal.html` en tu navegador web de preferencia si no requieres un servidor local.
